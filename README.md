# Binary Tree Traversal Visualizer (Menu Driven)

## Category
Data Structures - Trees (ADSA Subject)

## Description
This project visualizes binary tree operations using a menu-driven interactive program.
Users can insert nodes, display the tree structure, and view Preorder, Inorder, and Postorder traversals.

## Features
- Insert nodes into a Binary Search Tree (BST)
- View tree structure in ASCII visual form
- Perform three main tree traversals:
  - Preorder
  - Inorder
  - Postorder
- Easy-to-use menu interface

## How to Compile & Run
```
g++ binary_tree_menu_visualizer.cpp -o tree
./tree
```
(or run using VS Code Run/Debug buttons)
